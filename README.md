Repository Description

This repository contains homework assignments and coding exercises completed as part of the Machine Learning course. The tasks focus on implementing mathematical algorithms and numerical approximations using Python in a Jupyter Notebook environment.

Problem Details

Problem 1: Euclid's Algorithm
This problem involves implementing Euclid's Algorithm to compute the Greatest Common Divisor (GCD) of two integers.
Tasks included:
* Implementing a Python function gcd(m, n) to compute the greatest common divisor.
* Using the classical Euclidean algorithm which repeatedly applies the modulus operation until the remainder becomes zero.
* Understanding how the algorithm behaves when negative numbers are provided as input.


Problem 2: Approximating Euler's Number (e)
This problem explores numerical methods to approximate Euler's number (e).
Tasks included:
* Implementing a function euler_limit(n) that approximates ( e ) using the limit:
* Observing the effect of very large values of n and discussing floating-point precision issues.
* Implementing another function euler_sum(n) to approximate ( e ) using the infinite series:
* Computing the first n terms of the series to estimate the value of ( e ).



Problem 3: Testing Properties of Integers
This problem focuses on determining whether a number is a power of another number.
Tasks included:
* Implementing a function is_power_of_2(n) that determines if a number is a power of 2 using division and modulus operations.
* Generalizing the function to is_power(b, n) which checks whether n is a power of b.
* Practicing conditional statements and logical reasoning used in algorithm design.
